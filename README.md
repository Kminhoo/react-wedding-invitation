# 📑 프로젝트 소개

개인 프로젝트로 진행한 모바일 청첩장 서비스입니다. 기획부터 개발, 배포까지 2주간 집중 개발하여 사용자 경험과 목적에 맞춰 모바일 서비스를 구현했습니다.

<br>

# 🕹️ 주요 기능
- 🗺️ Kakao 지도 API를 활용한 식장 위치 정보 제공
- 📋 Firebase 기반 참석 여부 기능
- ✍️ 방명록 CRUD 기능
- 📲 카카오톡 공유하기 API 연동
- 🚨 React-Error-Boundary를 활용한 비동기 에러 핸들링

<br>
